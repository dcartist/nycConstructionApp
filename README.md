# nycConstructionApp


## About this Repo
DOB Job Application Filing

This dataset contains all job applications submitted through the Borough Offices, through eFiling, or through the HUB, which have a "Latest Action Date" since January 1, 2000.



## Technology Used
 * javescript
 * ExpressJs
 * Heroku
 * Mongoose/Mongo DB
 * Cors
 * Body-Parser
